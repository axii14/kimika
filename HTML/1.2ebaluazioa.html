    Kuantu zenbakiak

n: kuantu zenbaki nagusia. Orbitalaren tamainarekin erlazionatuta dago. n= 1, 2, 3...
l: kuantu zenbaki azimutala. Orbitalaren formarekin erlazionatuta dago. l= 0, 1, 2...     BETI l = n-1
    Balio bakoitzak azpimaila bat adierazten du:

l = 0 ⤍ s orbitala

l = 1 ⤍ p orbitala

l = 2 ⤍ d orbitala

l = 3 ⤍ f orbitala
