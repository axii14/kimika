Kimika organikoa

    Sarrera

Konposatuek gutxienez C eta H atomo bat izan behar dute.

Atomo:
    - primarioak: C atomo bakarra.
    - sekundarioak: bi C atomo.
    - tertziarioak: hiru C atomo.
    - kuaternarioak: lau C atomoa.




    Isomeria

Koposatu batzuek atomo eta kopuru berdinak dituzte baina substantzia desberdinak dira. Substantzia horiei isomeroak deitzen zaie, eta bereizteko arau batzuk kontuan izan behar dira. Bi talde nagusitan, eta hauen barruan beste hainbat taldeetan banatzen dira.

    Talde funtzionalak eta serie homologoak

Ezaugarri kimiko berdinak duten atomo taldeei talde funtzionalak deitzen zaie. Eta talde berekoak diren baina karbono kantitate desberdinak duten konposatuak serie homologoak osatzen dute.

Hidrokarburoak:

Alkanoak C-C:

Atomoak - bakarrarekin lotzen dira. -Ano atzikia dute eta katea osorik ez badago,beste zerbaiteri lotuta badago, erradikal deitzen zaio.

Alkenoak C=C:

Atomoak - bikoitzarekin lotzen dira. -Eno atzikia dute eta lotura bikoitza BETI atomo txikienen artean jartzen da.

Alkinoak C≡C:

Atomoak - hirukoitzarekin lotzen dira. -Ino atzikia dute.

Hidrokarburo ziklikoak:

Kate itxiak dira. Ziklo- aurrizkia dute.

Deribatu halogenatuak:

Aurrekoaren berdinak dira, baina hidrokarburoak halogeno (Cl, Br, I) itsatsita dauka.

Hidrokarburo aromatuak:

6 C atomoz osatutako kate itxiak dira. -Bentzeno atzizkia dute.

Konposatu oxigenatuak:

Alkoholak R-OH:

-ol atzizkia dute eta OH atomo txikienareri lotzen zaio-

Fenolak ArOH:

Aurrekoaren berdina da baina + bentzenoa. -Bentzeno atzikia dute.

Tiolak R-SH:

-tiol atzizkia dute.

Esterrak R-O-R-:

Bi erradikalen batura dira. -eter atzizkia dute.

Aldehidoak R-HCO:

Hidrokarburoak -al atzizkia dauka.

Zetonak R-C=O-R':

-ona atzizkia dute.

Azidoak R-COOH:

Azido deitzen dira -oiko atzizkia dute.

Konposatu nitrogenatuak:

Aminak R-NH2:

Erradikalak dira eta -amina atzizkia dute.

Amidak RCONH2:

Erradikalak dira eta -amida atzizkia dute.

Nitrioak R-CN:

Erradikalak dira eta -nitriloa atzizkia dute.

Nitrokonposatuak R-NO2:

Kate karbonatuak dira NO2-ri lotuta. Aurretik nitro jartzen zaie eta erradikalak dira.
